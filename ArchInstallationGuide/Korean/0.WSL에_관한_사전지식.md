## WSL(Windows Subsystem for Linux)란 무엇인가?
WSL, Windows Subsystem for Linux 는 윈도우10 이상의 기기에서 


## WSL로 아치 리눅스 설치하기
윈도우에서 CMD(커맨드 프롬프트) 또는 PowerShell 를 실행한 후 다음 순서대로 입력한다.
```
wsl --install --no-distribution
wsl --list --online              
wsl --install archlinux        
```

다음 명령어를 통해 설치 여부를 확인할 수 있다.
```
wsl --list --verbose 
```
목록에 archlinux 항목이 보인다면, 정상적으로 설치된 것이다.

다음 명령어로 WSL 에 설치한 아치 리눅스를 실행한다.
```
wsl --distribution archlinux
```

실행 이후 다음과 같이 명령어 입력 창이 뜬다. 이는 윈도우에서 실행한 CMD 또는 PowerShell 내부에 
```
[Username]@[Hostname Directory]# :

[root@wsl admin]# :
```

명령어에 대한 옵션을 다음의 경우 축약형으로 사용할 수 있다.
```
--distribution -> -d
--list -> -l
--online -> -o
--verbose -> -v
--terminate -> -t

wsl --list --online           ->  wsl -l -o
wsl --list --verbose          ->  wsl -l -v
wsl --distribution archlinux  ->  wsl -d archlinux
```


## WSL 아치 리눅스 초기 설정
**sudo**
```
pacman -Syu
pacman -S sudo nano

EDITOR=nano visudo
```

열린 문서 내에서 다음 부분을 찾아 수정한다. 
```
# %wheel ALL=(ALL:ALL) ALL

%wheel ALL=(ALL:ALL) ALL
```
Ctrl + x -> y -> Enter 이 순서로 저장 후 종료한다.

**User Account**
WSL에서 아치 리눅스는 기본 사용자 계정이 없는 채로 설치되므로, 명령어를 통해 사용자 계정을 생성할 수 있다.
```
sudo useradd -m -G wheel -s /bin/bash [username]
sudo passwd [username]

sudo vim /etc/wsl.conf
```
- [username] 를 원하는 사용자명으로 대치하여 입력한다.

/etc/wsl.conf 파일을 다음과 같은 내용을 추가한다.
```
[user]
default=[username]

[network]
hostname=[hostname]
generateHosts=false
```
- [username] 를 원하는 사용자명으로 대치하여 입력한다.
- [hostname] 를 원하는 이름으로 대치하여 입력한다.
