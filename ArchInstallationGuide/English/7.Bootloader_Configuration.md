At the end of the installation, there will be three options.
`Exit archinstall` 
`Reboot system` 
`Chroot into installation`

You can check the GRUB(boot loader) status with the 'Chroot' option.
`lsblk -f`
`fdisk -l`
`ls /boot/efi/EFI`

`cat /etc/fstab` `/` `/boot` `/boot/efi`
`ls /boot` `vmlinuz-linux` `initramfs-linux.img` `/boot/grub/`
`ls /boot/efi/EFI/GRUB` `grubx64.efi`
`cat /boot/grub/grub.cfg | less`
`mount | grep efi`

If there is NOT boot loader file, you can install it with this command.
`grub-install --target=x86_64-efi --efi-directory=/boot/efi --bootloader-id=GRUB --boot-directory=/boot --recheck`
`grub-mkconfig -o /boot/grub/grub.cfg`
`efibootmgr`

efibootmgr --create \
  --disk /dev/sdX \          # whole disk (e.g. /dev/sda)
  --part N \                 # EFI partition number (e.g. 1)
  --label "Arch Linux" \
  --loader /EFI/GRUB/grubx64.efi
