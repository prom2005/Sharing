## 2-3. Network Configuration

If you download arch linux using **Virtual Machine** like virtualbox or VMware, Network will be autometically connect.

| Command                     | Means                      | Additionally...              |
| --------------------------- | -------------------------- | ---------------------------- |
| iwctl                       | get into network connector |                              |
| device list                 |                            |                              |
| station wlan0 scan          |                            |                              |
| station wlan0 get-networks  | show wifi list             |                              |
| station wlan0 connect $SSID | connect                    | $SSID will be your wifi name |


Then, arch installer ask you a password. Just type password and `exit`
And, you can check network using this command. `ping -c 3 archlinux.org`
