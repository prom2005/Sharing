## 패키지 설치
```
sudo pacman -S [Package]
sudo pacman -S git wget curl
```


## 패키지 관리
```
sudo pacman -Syu

sudo pacman -Rns $(pacman -Qtdq)
sudo pacman -Scc

sudo pacman -Qs [Package]
sudo pacman -Qq
```


## AUR이란?
AUR, Arch User Repository 란

**yay 설치**
```
sudo pacman -S --needed git base-devel
git clone https://aur.archlinux.org/yay.git
cd yay
makepkg -si
cd ..
rm -rf yay

yay -Syu
```

**yay 패키지 관리**
```
yay -S [Package]

yay -Qs [Package]
yay -Qq

yay -Rns [Package]
yay -Scc
```
